Errors & Validation
===================

	If a request fails due to client error, the resource will return an HTTP response code in the 40x range. These can be broadly categorised into: 
	
	+-------------------+------------+
	|HTTP Code          |Description |
	+===================+============+
	|400 (Bad Request)  |            |
	+-------------------+------------+
	|401 (Unauthorized) |            |
	+-------------------+------------+
	|403 (Forbidden)    |            |
	+-------------------+------------+
	|404 (Not Found)    |            |
	+-------------------+------------+
	|405 (Method Not    |            |
	|Allowed)           |            |
	+-------------------+------------+
